## [0.1.2] - any-edit - 2026-01-17

### Added

- 'edit' subcommand takes optional --doc url, as substitute for space_id and object_id. The url is in the format generated by the app menus "Copy Link" or "Copy Deeplink".
- if --url is not provided, first checks for environment variable ANYTYPE_URL, then defaults to http://127.0.0.1:31009 (desktop app)

### Changed

- new command 'any-edit edit-current' replaces 'any-edit edit --current'
- bump dependencies; anytype v0.2.9

## [0.1.1] - any-edit

### Changed

- Use rustls (native roots) for HTTP TLS to avoid OpenSSL install errors.
- Uses anytype-v0.2.8.
